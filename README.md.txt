### Core Architecture Features
* Full Flutter Material 3 design layout tailored for mobile displays.
* Multi-habit scheduling with Boolean logic tracking arrays.
* Visual progress calculators tracking structural routine performance scores.

### How to Compile Locally
1. Install Flutter SDK on your system.
2. Run `flutter pub get` to restore project dependencies.
3. Run `flutter run` with an emulator running to execute the mobile application.
